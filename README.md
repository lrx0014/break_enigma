# break_enigma
algorithm implementation for turing bombe machine

## HOW-TO

### Use Colab
https://colab.research.google.com/drive/1NzkP7RSpoMQddvyRfgcluQrDAsdX7IoX

### Local Run

install dependencies
```shell
python3 -m pip install pyenigma==0.4.0
```

then run the script.
```shell
cd src
python run_batons.py 
python run_bombe.py
```
